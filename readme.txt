== Description ==

[WP Sentence](http://jakubas.net.pl/projekty/narzedzia/wp-sentence-plugin) is a wordpress plugin that allows you to add new widget to Your sidebar. 
WP Sentence shows one of the 470 citations. 
When the page refreshes widget show different (random) quote.

List of all quotes:

[WP Sentence - Quotes List](http://jakubas.net.pl/projekty/narzedzia/wp-sentence-plugin/wp-sentence-plugin-quotes-list)

== Installation ==

    * Download it 
    * Extract all files from the zip archive
    * Copy the wp-followme folder to wp-content/plugins
    * Activate the plugin through the �Plugins� menu in wordpress
    * Add WP Sentence Widget to your sidebar
    * That's all !
